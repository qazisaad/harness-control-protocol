# License Decision

Decision: Apache License 2.0.

Rationale: Harness Control Protocol is intended to be a public protocol and local runner stack that other applications, companies, and open-source projects can adopt. Apache-2.0 is permissive while also including an explicit patent grant and patent retaliation clause, which is valuable for protocol and infrastructure projects.
